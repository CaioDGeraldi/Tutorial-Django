# Sobre o Projeto
Este tutorial foi desenvolvido para consolidar os conhecimentos adquiridos sobre **Django**, cobrindo desde a instalação até a estruturação de uma aplicação web funcional.

### O que você vai encontrar:
- Configuração de Ambiente Virtual (`venv`).
- Estrutura de Pastas e Arquivos.
- Comandos Essenciais (`manage.py`).
- Integração com HTML/CSS.

<div id="pt-steps"></div>

### - *Passo a Passo Base*
1. **Instalação:** `pip install django`
2. **Criação do Projeto:** `django-admin startproject nome_projeto`
3. **Servidor:** `python manage.py runserver`
<br>...aprendizado baseado nas aulas da ETEC Alberto Feres e documentação oficial.
